# CasaWelt - Real Estate B2B SaaS

CasaWelt is a complete, professional B2B SaaS platform for real estate agencies. It includes a CRM, property catalog, booking system, billing, advanced reports, and more.

## What you received
The online version you are viewing is a **Demo** running entirely in the browser using local state. This allows you to test all features without needing a backend. 

However, the code you have downloaded is the **complete, production-ready application**. It includes real integrations for Supabase (Database/Auth), Stripe (Payments), and SendGrid (Emails). To make the app real, you just need to connect your own services.

## Requirements
- Node.js 18+
- A Supabase account (https://supabase.com)
- A Stripe account (https://stripe.com)
- A SendGrid account (https://sendgrid.com)
